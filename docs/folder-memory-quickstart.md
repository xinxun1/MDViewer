# 🚀 快速体验文件夹记忆功能

## 第一次使用

1. **打开应用**  
   用 Chrome/Edge 浏览器打开 `standalone.html`

2. **选择文件夹**  
   点击左上角的"打开文件夹"按钮，选择你的 Markdown 文件夹

3. **开始工作**  
   系统会自动保存这个文件夹，正常浏览和编辑你的文件

## 第二次使用

1. **直接打开**  
   再次打开 `standalone.html`

2. **自动加载** ✨  
   看到提示：
   ```
   ✅ 已自动打开上次的文件夹: [你的文件夹名]
   ```

3. **继续工作**  
   文件列表自动加载，直接继续上次的工作！

## 权限请求

如果浏览器弹出权限确认框：

```
[浏览器提示]
standalone.html 想要查看 "[文件夹名]" 中的文件
[允许] [拒绝]
```

**点击"允许"**即可恢复上次的文件夹！

## 注意事项

⚠️ **隐私模式** - 隐私模式下关闭浏览器会清除记录  
⚠️ **清除数据** - 清除浏览器数据会清除文件夹记录  
✅ **更换文件夹** - 打开新文件夹会自动替换旧记录
