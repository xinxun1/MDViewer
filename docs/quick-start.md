# MD Viewer 快速开始指南

## 🎯 如何使用

### 1. 打开文件夹

点击左侧的 **"打开文件夹"** 按钮，选择包含 Markdown 文件的文件夹。

浏览器会请求访问权限，请点击"允许"。

### 2. 浏览文件

文件树会自动显示该文件夹下的所有 `.md` 和 `.markdown` 文件。

- 📁 文件夹会显示为黄色图标
- 📄 Markdown 文件显示为蓝色图标
- 点击文件夹可以展开/收起
- 点击文件即可查看内容

### 3. 搜索文件

使用顶部的搜索框快速查找文件，支持文件名搜索。

### 4. 查看模式

**三种查看模式：**

- 👁️ **预览模式** - 只读查看，适合阅读
- ✏️ **编辑模式** - 纯编辑界面，专注写作
- 📄 **分栏模式** - 左侧编辑，右侧实时预览

### 5. 编辑和保存

1. 切换到编辑模式或分栏模式
2. 在编辑器中修改内容
3. 按 `Ctrl + S` 或点击"保存"按钮保存

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存当前文件 |

## 🎨 支持的 Markdown 格式

### ✅ 基础语法

- 标题（h1-h6）
- 粗体、斜体、删除线
- 有序/无序列表
- 任务列表（复选框）
- 链接和图片
- 引用块
- 水平分割线

### ✅ 代码

- 行内代码
- 代码块（支持 180+ 种语言高亮）
- 语法：\`\`\`语言名称

### ✅ 表格

支持表格和文本对齐：

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容 | 内容 | 内容 |
```

### ✅ 数学公式（KaTeX）

**行内公式：** `$E=mc^2$`

**块级公式：**
```markdown
$$
\frac{-b \pm \sqrt{b^2-4ac}}{2a}
$$
```

### ✅ 流程图（Mermaid）

```markdown
\`\`\`mermaid
graph TD
    A[开始] --> B[处理]
    B --> C[结束]
\`\`\`
```

支持的图表类型：
- 流程图（graph/flowchart）
- 序列图（sequenceDiagram）
- 甘特图（gantt）
- 类图（classDiagram）
- 状态图（stateDiagram）
- 饼图（pie）
- Git 图（gitGraph）

## 🌙 主题切换

点击右上角的 🌙/☀️ 图标切换深色/浅色主题。

主题设置会自动保存。

## 💡 使用技巧

### 1. 文件组织

建议按项目或主题组织文件夹：

```
项目文档/
├── 设计/
│   ├── UI设计.md
│   └── 架构设计.md
├── 开发/
│   ├── API文档.md
│   └── 数据库设计.md
└── README.md
```

### 2. 实时预览

使用分栏模式可以边写边看效果，特别适合：
- 编写复杂表格
- 调试数学公式
- 绘制流程图

### 3. 代码高亮

在代码块中指定语言名称即可获得语法高亮：

\`\`\`javascript
const hello = 'world';
\`\`\`

常用语言：
- `javascript`, `python`, `java`, `c`, `cpp`
- `html`, `css`, `sql`, `bash`, `json`
- `markdown`, `yaml`, `xml`

### 4. 任务管理

使用任务列表管理待办事项：

```markdown
- [x] 已完成的任务
- [ ] 待完成的任务
```

## ⚠️ 注意事项

### 浏览器要求

**支持的浏览器：**
- ✅ Chrome 86+
- ✅ Edge 86+
- ❌ Firefox（不支持 File System Access API）
- ❌ Safari（不支持 File System Access API）

### 权限说明

首次打开文件夹时，浏览器会请求以下权限：
- 📂 读取文件夹内容
- 📝 读写文件

这些权限仅限于您选择的文件夹，不会访问其他位置。

### 性能提示

- 大文件（>1MB）可能加载较慢
- 包含大量图表的文档渲染需要时间
- 建议文件大小控制在合理范围内

## 🔧 常见问题

### Q: 无法打开文件夹？

**A:** 确保使用 Chrome 或 Edge 浏览器，版本 86 以上。

### Q: 保存失败？

**A:** 检查文件是否被其他程序占用，或文件夹是否有写入权限。

### Q: 数学公式不显示？

**A:** 刷新页面，确保 KaTeX 库加载完成。

### Q: 流程图显示错误？

**A:** 检查 Mermaid 语法是否正确，参考 [Mermaid 文档](https://mermaid.js.org/)。

### Q: 图片不显示？

**A:** 
- 检查图片路径是否正确
- 使用相对路径引用同文件夹下的图片
- 或使用完整的网络图片 URL
